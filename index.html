Project Title: Detection and Recognition of Visual Objects and Biometrics

1. Overview

This project implements real-time computer vision capabilities in a web browser. It leverages pre-trained Deep Learning models to identify human faces, fruits, and simulate biometric ridge extraction.

2. Technologies

  • TensorFlow.js (Neural Network Engine)
  • COCO-SSD (Model Architecture)
  • HTML5 Canvas/Media Stream (Video Input)
  • Tailwind CSS (Responsive UI Design)

3. Deep Learning Methodology

The system uses Transfer Learning by loading the COCO-SSD model, which has been trained on millions of images. For fingerprints, the application simulates the Feature Extraction layer of a Siamese Network by applying grayscale normalization and thresholding to highlight ridge patterns (minutiae).
