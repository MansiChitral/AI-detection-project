Mini Project Report

Project Aim

Detection and recognition of objects (Face, Fruit) and Biometrics (Fingerprint) using Deep Learning.

Technology Used

JavaScript, TensorFlow.js, COCO-SSD Model, Tailwind CSS.

System Methodology

The application uses TensorFlow.js to run a pre-trained SSD (Single Shot MultiBox Detector) model on the client side. This ensures low latency and high privacy. The fingerprint module utilizes image binarization and thresholding techniques to highlight minutiae patterns for biometric feature mapping.
