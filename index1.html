Project Implementation Details

1. Technology Stack

  • Frontend: HTML5, Tailwind CSS
  • AI Engine: TensorFlow.js
  • Models: COCO-SSD (Object Detection)
  • API: MediaDevices (Camera Access)

2. Methodology

The system utilizes Transfer Learning. COCO-SSD is a pre-trained model that extracts features using a Single Shot MultiBox Detector architecture. For fingerprints, we implement Image Binarization, a crucial preprocessing step for Minutiae Extraction in biometric recognition.

3. Key Features

  • Real-time Latency: ~50-100ms
  • Zero-Server Processing (Client Side)
  • Multi-object concurrency
  • Edge-detection for biometrics

Academic Note for Viva:
